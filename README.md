🌐 Personal Profile Website – README
📌 Overview

このプロジェクトは、HTML / CSS / JavaScript を用いて制作する個人プロフィールサイトです。
名前・所属だけで終わらない、「あなたの魅力・経験・技術」がしっかり伝わる ポートフォリオ兼プロフィールページ を目指しています。

🎯 Purpose

GitHub のプロフィール代わりになる Web ページを公開する

自分のスキル・制作物・活動歴を分かりやすく見せる

シンプルだけど今っぽい UI / UX を意識した構成にする

就活・技術ポートフォリオとしても活用できるようにする

🎨 Design Concept (CSS)

配色：白・薄グレーをベースに青 or 紫をアクセント

フォント：見出し大きめ、本文は読みやすいサイズ

カードデザイン：軽いシャドウ＋ホバーで浮くアニメーション

レスポンシブ対応：max-width を設定して中央寄せ

余白設計：セクション間に十分なスペースを確保
